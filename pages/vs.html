This is my first solo project. This system is used by CEDC (COMELEC EMPLOYEES DEVELOPMENT COOPERATIVE). This project is created September of 2017 and used March 23 2018.

The system use a lan connection, it can accummulate data in real-time and create report after the voting or event.

Vote in any computer (Connected on the LAN) - you can vote in any computer because the system automatically segregates votes for each district.

Can create report - the system can create report for every district votes and the over all votes.
